# Honolulu Blues Fantasy Football SPA

A single-page Angular application designed to import weekly fantasy football scores and generate rich, data-driven insights for your league.
The project emphasizes clarity, fast iteration, and clean UX using Angular Material.

This repository is organized for incremental versioned development.
All architecture notes, feature specifications, analytics formulas, and release documentation are maintained in the `/docs` directory.
